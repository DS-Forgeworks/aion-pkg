# Aion — Intelligent Outreach Platform

Find leads. Enrich contacts. Send campaigns. All from your machine.

## One-Line Install
```bash
curl -fsSL https://dsforgeworks.com/aion/install.sh | bash
```

## What it does
- **Find leads** — scrape websites, directories, social profiles
- **Enrich contacts** — fill in emails, phone numbers, social links
- **Kanban board** — drag contacts from "sourced" to "contacted"
- **Send campaigns** — Gmail-powered, with daily limits and reply monitoring

## Requirements
- Node.js 18+
- macOS or Linux
- A Google account (for Gmail sending)

## Manual Setup
```bash
npm install -g aion-outreach
aion setup
aion start
```
