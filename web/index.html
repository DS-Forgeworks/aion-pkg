Gmail API Setup

Aion sends emails through Gmail API. Follow the steps below to enable it.

  1. Go to Google Cloud Console and create a new project
  2. Enable the Gmail API from the API Library
  3. Go to Credentials → Create Credentials → OAuth 2.0 Client ID
  4. Application type: Desktop app
  5. Add redirect URI: http://localhost:7443/oauth/callback
  6. Copy your Client ID and Client Secret
  7. Run aion setup in the terminal to complete authentication
